# UNIMAR - Sistema de Cuestionarios y Evaluaciones

Este repositorio contiene el prototipo funcional de la Plataforma de Evaluaciones para la **Universidad de Margarita (UNIMAR)**. El sistema está desarrollado con un backend robusto en **Flask** y **SQLite**, complementado con una interfaz de usuario dinámica y moderna de alto contraste optimizada mediante **Tailwind CSS**.

El proyecto incluye control de roles (Profesores y Estudiantes), persistencia de datos segura (idempotente) y diseño responsivo premium.

---
